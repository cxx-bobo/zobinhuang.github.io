Locally Weighted Regression

    回顾我们在 线性回归和梯度下降 一文中介绍的 Linear Regression (LR) 算法，我们基于 LR 可以在 Training Set 上拟合出来一条直线，用于对新数据进行预测，如上图所示。但是倘若我们的样本空间中的数据并不能很好地使用一条直线来进行拟合的话，我们使用 LR 算法进行预测的结果将是十分糟糕的。

    本小节介绍一种针对 LR 算法进行优化的方法，Namely Locally Weighted Regression (LWR, 局部加权回归)。

    回顾我们上一篇文章使用 LR 解决问题时定义的 Lost Function:

`\mathcal{J}(\Theta) = \frac{1}{2}\sum_{i=0}^{m}(h(X^{(i)};\Theta)-Y^{(i)})^2`

    LWR 的基本思路是: 对新数据 `X^{(n ew)}` 进行预测时，首先在 Training Set 中找到与 `X^{(n ew)}` 更加接近的 Samples，基于这些 Samples 使用 LR 算出来一组参数 `\Theta^{(s)}`，然后使用 `\Theta^{(s)}` 完成对 `X^{(n ew)}` 的预测。

    基于上述思路，我们修改我们的 Lost Function 如下所示:

`\mathcal{J}(\Theta) = \frac{1}{2}\sum_{i=0}^{m}``w^{(i)}``(h(X^{(i)};\Theta)-Y^{(i)})^2`

    其中的 `w^{(i)}` 定义如下:

`w^{(i)} = exp(-\frac{[X^{(i)}-X^{(n ew)}]^2}{2\tau^2})`

    `w^{(i)}` 的含义是：对于 Training Set 中的各条 Samples `(X^{(i)},Y^{(i)})`，距离 `X^{(n ew)` 越近的 Sample 的权值 `w^{(i)}` 将更接近于 1，距离 `X^{(n ew)` 越远的 Sample 的权值 `w^{(i)}` 将更接近于 0。这样一来，Training Set 中的各条 Samples 对最终训练出来的参数的影响就有区别了。

Pobabilistic Interpretation

    在上一篇文章 线性回归和梯度下降 中，我们定义了 1 的 Lost Function，在本节中我们从概率的角度将说明为什么将 Lost Function 定义为这样是合理的。

    首先我们假设，在 Training Set 中，各条 Sample 的 Label 和 Features 存在如下关系：

`Y^{(i)}=\Theta^TX^{(i)}+\epsilon^{(i)}`

    其中 `\epsilon^{(i)}` 是一个误差项，是一个随机变量，描述了 Unmodel Effects，也即我们没办法在我们的 LR 模型上进行描述的关系。我们进一步假设 `\epsilon^{(i)}` 是 Independently and Identically Distributed (i.e. IID, 独立同分布) 的，并且服从的是均值为 `0`，方差为 `\sigma^2` 的正态分布，`\epsilon^{(i)}~N(0,\sigma^2)`，也即:

`p(\epsilon^{(i)})=\frac{1}{\sqrt{2\pi}\sigma}exp(-\frac{(\epsilon^{(i)})^2}{2\sigma^2})`

    这暗示了，对于第 `i` 条 Sample，在已知 Features `X^{(i)}` 和给定参数 `\Theta` 的情况下，Label `Y^{(i)}` 同样是一个服从正态分布的随机变量，其以 `2\sigma^2` 为方差在 `\Theta^TX^{(i)}` 附近波动，也即满足如下关系:

`p(Y^{(i)}|X^{(i)};\Theta) = \frac{1}{\sqrt{2\pi}\sigma}exp(-\frac{(Y^{(i)}-\Theta^TX^{(i)})^2}{2\sigma^2})`

    也即：` (Y^{(i)}|X^{(i)};\Theta) ~ N(\Theta^TX^{(i)},2\sigma^2)`。

    好了，现在让我们思考我们的初衷: 我们的目的是选取一组参数 `\Theta`，使得给予我们的模型预测的结果 `\Theta^TX^{(i)}` 更加接近我们的 Label `Y^{(i)}`，也就是说 `\Theta` 是一个我们希望调整的量，而 `X^{(i)}` 和 `Y^{(i)}` 对于我们来说是定量。在上面的分析中，我们一直基于给定的 `\Theta`，使用 「概率」这个概念来描述我们的系统。因此现在我们转换思路，把 `\Theta` 视为可变量来描述我们的系统，在数学上的相关概念是 Likelyhood (似然函数)。你可以通过阅读拓展博文 延伸阅读 —— 统计推断的基本原理和分类 来理解「似然」的概念。
